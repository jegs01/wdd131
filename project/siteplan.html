Site Plan for Remote Job Listing

Site Name

The site name is remotelisting.com. This name was selected because it effectively conveys the purpose of the website, which is to collect job listing information, particularly remote jobs.

Site Purpose

The site aims to collect job listing information in various remote job categories, allowing employers to post remote job vacancies.

Scenarios

Scenario 1: A Human Resource Manager that wants to find out more information about how to navigate the site, can do this via the Contact Us form.

Scenario 2: A company needing to hire a remote customer support representative posts a job listing on the site to reach potential candidates.

Color Schema

The color scheme for the website includes:

  • Primary Color: #35424a (dark slate gray) - This will be used for the header, navigation bar, and headings.
  • Accent Color: #f4f4f4 (light gray) - This will be used for the background and content sections to ensure readability and a clean look.

Typography

The typography for the website includes:

  • Primary Font: Arial, sans-serif - This will be used for body text for readability.
  • Secondary Font: 'Helvetica Neue', Helvetica, Arial, sans-serif - This will be used for headings to provide a clean and modern look.
